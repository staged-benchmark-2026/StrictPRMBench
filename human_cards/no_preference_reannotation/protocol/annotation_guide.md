# Annotation Guide: No-Preference Forced-Choice Replication

You will see a math problem and two candidate reasoning traces, Trace A and Trace B.

Choose exactly one of:

- A: Trace A is clearly better.
- B: Trace B is clearly better.
- No preference: the two traces are effectively tied in correctness, clarity, completeness, and auditability.

Use No preference when neither trace is meaningfully better, even if the traces differ in length or wording.
Do not try to infer which trace came from which system or selection rule. Judge only the visible reasoning.

Leave notes only when useful, for example if both traces appear wrong, both are ambiguous, or the comparison is unusually difficult.
